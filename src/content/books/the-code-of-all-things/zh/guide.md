本书把“世界由规则驱动”作为一个工作假设，用代码思维重新组织普通人理解 AI、文明、生命和社会协作的方式。

它不是要宣称某种终极真理，而是提供一套可以迭代的认知坐标系：当我们给 Agent 下指令、设计自动化执行流、判断技术风险时，先问它是否顺着更底层的规则运行。

## 阅读顺序

建议先读序言、第一章和第二章：先理解自然语言为什么正在变成新的“可执行入口”，再理解为什么所有认知模型都只能不断逼近底层规则。随后读目录，按四篇结构继续推进：规则坐标系、文明反编译、创世沙盘、大道对齐。

## 当前发布说明

博客版已发布导读、目录、序言、第一章和第二章。后续章节会按同一套中英文对应结构逐步整理上线，避免把未经校对的全文粗糙堆进站点。

## 核心框架

- 物理主规则：所有系统必须服从的宇宙底层约束。
- 生命主规则：碳基生命维持稳态、生存和延续的原生脚本。
- 社会协作规则：群体为了降熵生存而形成的法律、伦理、货币和制度协议。
- 符号与认知规则：人类用语言、数学、模型对现实进行降维映射的方式。
- Agent 执行规则：我们在智能时代用自然语言写下的目标、边界、权限和执行逻辑。
